# GitHub Pages 部署说明

## 如何启用 GitHub Pages

1. 访问仓库设置：https://github.com/jackegg123/agentJob/settings/pages
2. 在 "Build and deployment" 部分：
   - Source: 选择 `Deploy from a branch`
   - Branch: 选择 `main` 分支，文件夹 `/ (root)`
3. 点击 Save

## 访问链接

启用后，可以通过以下链接访问：

- **主页**: https://jackegg123.github.io/agentJob/
- **About Mushroom**: https://jackegg123.github.io/agentJob/about-mushroom/
- **Claude Code Demo**: https://jackegg123.github.io/agentJob/claude-demo/

## 注意事项

- 首次部署可能需要 1-2 分钟生效
- 确保仓库是公开的（public）才能通过 GitHub Pages 访问
- 如果遇到 404，检查 Pages 设置是否正确保存
